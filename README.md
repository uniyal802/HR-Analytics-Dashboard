# HR Analytics Dashboard

This project is an interactive HR Analytics Dashboard built to analyze employee data and generate meaningful insights for HR decision-making. It helps organizations understand workforce patterns such as attrition, demographics, performance, and engagement.

# 🎯 Objective
🚀 Overview

The HR Analytics Dashboard is an interactive data analytics application designed to analyze employee data and generate meaningful HR insights.

It helps organizations understand workforce structure, attrition patterns, salary distribution, and employee demographics through an interactive dashboard.

# 🎯 Objective
Analyze employee attrition trends
Identify key factors affecting employee retention
Understand department-wise workforce distribution
Visualize salary, age, and performance patterns
Provide interactive HR insights for decision-making

# Tech Stack
Python:- Core programming language used for data processing and logic building.

Pandas:- Used for data cleaning, filtering, grouping, and aggregation of HR datasets.

NumPy:-Used for numerical operations and efficient handling of structured data.

Streamlit:- Converted Python scripts into an interactive web-based dashboard.

Matplotlib / Plotly:- Used for data visualization and generating interactive charts.

# 🚀 Getting Started
1. Clone the Repository
git clone https://github.com/uniyal802/HR-Analytics-Dashboard.git

2. Install Dependencies
   pip install -r requirements.txt

3. Run the App
   streamlit run app/app.py

# Key Insights Generated
- Employees in specific departments show higher attrition rates
- Salary distribution varies significantly across job roles
- Age group 25–35 shows highest workforce concentration
- Performance ratings correlate with tenure in the company
