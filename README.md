# HR Analytics Dashboard

This project is an interactive HR Analytics Dashboard built to analyze employee data and generate meaningful insights for HR decision-making. It helps organizations understand workforce patterns such as attrition, demographics, performance, and engagement.

# 🎯 Objective

To convert raw HR data into visual insights that help in:

Understanding employee attrition trends
Analyzing department-wise workforce distribution
Identifying key factors affecting employee retention
Supporting data-driven HR decisions
⚙️ Tech Stack
Python
Pandas (data cleaning & analysis)
NumPy
Streamlit (dashboard UI)
Plotly / Matplotlib (data visualization)
📊 Key Features
Employee data cleaning and preprocessing
Attrition analysis by department, gender, and age group
Salary and performance distribution insights
Interactive dashboard filters
Visual charts for better decision-making
Real-time data exploration using Streamlit UI
